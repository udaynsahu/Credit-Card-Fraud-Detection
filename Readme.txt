Simple Steps to execute this project:
[NOTE:- Make sure flask is installed properly in your system. If not, please install flask as well as all the supported libraries.] 

1) Open the folder where flask application and all it's allied libraries and applications are stored. Store all the above files in the folder.
2) Also, you need to fetch 3 more files. The names of the files are rf_model.pkl, fraudTest.csv, fraudTrain.csv. The Links to all the 3 files are given below.
3) Now, create a folder named "templates". Paste the Credit_Card.html file in the "templates" folder. 
4) Open Command Prompt or any other terminal and navigate to the folder where the flask application and all your files are stored.
5) Type "flask run" in the terminal.

Open the port mentioned in the terminal and the project is executed!!

rf_model.pkl :- https://drive.google.com/file/d/1G1VUcRsKgg5ctgaY9WmI-S0q_sw5WjIC/view?usp=sharing 
fraudTest.csv and fraudTrain.csv :- https://www.kaggle.com/datasets/kartik2112/fraud-detection
